# Express 10U Orange · Sun Prairie

Single-page weekend guide for the Evanston Express 10U Orange trip to the
Sun Prairie, WI tournament (July 10–12, 2026). Mobile-first, auto-scrolls
to the next event, with maps and the essentials for each stop.

Live site: https://paulsmith.github.io/sun-prairie/

The whole site is one file: `index.html`. Pushes to `main` deploy via
GitHub Actions (`.github/workflows/pages.yml`).
